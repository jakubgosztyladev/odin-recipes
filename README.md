# Odin Recipes

A simple recipe website built as part of The Odin Project curriculum.

## Description

This project demonstrates basic HTML skills including:

- Creating multiple linked HTML pages
- Using headings, paragraphs, lists, links, and images
- Organizing files in a logical directory structure
- Writing semantic HTML

## Recipes Included

- Beef Cheeseburger
- Homemade Pizza
- Grilled Salmon

## What I Learned

- HTML boilerplate structure
- Relative file paths and navigation
- Ordered and unordered lists
- Linking between pages
- Image embedding
- Basic Git workflow (add, commit, push)

## Technologies Used

- HTML5
- Git & GitHub
